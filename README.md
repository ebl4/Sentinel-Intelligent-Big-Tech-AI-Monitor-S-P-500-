# Sentinel: Intelligent Big Tech & AI Monitor (S&P 500)

**Sentinel** is a quantitative monitoring and value investing analysis platform focused on identifying pricing distortions, speculative AI bubble risks, and macroeconomic entry/exit points for major U.S. technology companies (**NVDA, PLTR, MSFT, AVGO**).

The application was built with a strong analytical approach, separating media narratives from the actual cash flows generated by companies and large-scale capital expenditures (**CAPEX**).

---

## 🚀 Core Features

* **Multidimensional Real-Time Scanning:** Direct integration with the Gemini 3.5-Flash API enhanced with real-time search tools (**Google Search Grounding**) to track updated earnings, YoY growth rates, and CAPEX metrics.

* **Stress Matrix & Simulation Engine:** Interactive dashboard sliders allow users to stress-test valuation multiples such as P/E (*Price-to-Earnings*), revenue growth, and Free Cash Flow Yield, automatically reclassifying asset risk levels.

* **AI Bubble Risk Thermometer (CAPEX):** Dedicated monitoring panel focused on correlating Big Tech CAPEX expansion with cash flow sustainability and GPU demand.

* **Quota-Resilient Architecture:** Elegant handling of network instability and critical `429 (Resource Exhausted / Quota Exceeded)` errors. In adverse scenarios, the system automatically activates a consolidated fallback dataset, ensuring the UI remains fully interactive.

* **Advanced Heuristic Charts:** Intuitive visualizations powered by Recharts comparing P/E multiples versus 5-year historical averages, alongside growth momentum and free cash flow generation.

* **Integrated Intelligence Console:** Enables qualitative analysis queries for additional AI-related S&P 500 assets.

---

## 🛠️ Tech Stack & Architecture

The application follows a modern Full-Stack ecosystem architecture (Express + React SPA powered by Vite):

* **Frontend:** React 19, TypeScript, Tailwind CSS, Lucide Icons, Recharts for data visualization, Plus Jakarta Sans & JetBrains Mono for immersive technical typography.

* **Backend (Express Server):** Implements a resilient backend proxy layer for Gemini SDK `@google/genai` API calls, ensuring secret API keys are never exposed to the client-side browser environment.

### Project Structure

* `server.ts` → Backend entry point. Handles resilient proxy requests and fallback historical stress dataset mechanisms.
* `src/App.tsx` → Main immersive dashboard containing the analytical console, pricing distortion charts, and fear/greed indicators.
* `src/types.ts` → Strict typing contracts for market data and search grounding structures.
* `src/components/MetricCard.tsx` → Asset profiling card with individual valuation simulation controls.
* `src/components/KnowledgeBaseDoc.tsx` → Human-readable knowledge base exposing structural market entry and exit strategies.

---

## 💎 Quantitative Decision Matrix & Risk Thresholds

Current asset classifications strictly follow these quantitative rules:

* **Nvidia (NVDA):** Critical warning triggered if Forward P/E exceeds 50x.

* **Palantir (PLTR):** Classified as a **Speculative Bubble** if current P/E surpasses 150x without corresponding revenue growth acceleration.

* **Magnificent 7 & Broadcom (AVGO):** Ideally positioned between 25x and 35x P/E. Above 40x requires a strong AI-driven operating margin expansion thesis.

* **CAPEX Indicator:** Nvidia and correlated semiconductor companies depend directly on infrastructure CAPEX expansion from MSFT, GOOGL, AMZN, and META. Any CAPEX contraction may trigger cascading downside pressure on Nvidia.

## 💻 Installation & Execution

### Prerequisites

Ensure you have Node.js v18+ installed.

### 1. Install dependencies

```bash

```

### 2. Configure your Gemini API key

Create a `.env` file based on `.env.example`:

```env

```

### 3. Run the local development environment

```bash
npm run dev
```

### 4. Build and run in standalone production mode

```bash
npm run build
npm start
```

# Sentinel: Monitor Inteligente de Big Techs & IA (S&P 500)

O **Sentinel** é um dispositivo de monitoramento quantitativo e análise de valor (*Value Investing*) focado em identificar distorções de preço, risco de bolhas especulativas em Inteligência Artificial e pontos macroeconômicos de entrada ou saída para algumas empresas americanas de tecnologia (**NVDA, PLTR, MSFT, AVGO**).

Esta aplicação foi construída com foco em rigor analítico, separando a narrativa midiática dos fluxos de caixa reais gerados pelas companhias e os investimentos de Capital em escala (**CAPEX**).

---

## 🚀 Funcionalidades Principais

- **Varredura Multidimensional Georreferenciada:** Integração direta com a API do Gemini 3.5-Flash habilitada com ferramentas de pesquisa em tempo real (**Google Search Grounding**) para rastrear lucros, taxas de crescimento YoY e CAPEX atualizados.
- **Matriz de Estresse e Simulação:** Controles deslizantes interativos no painel que permitem estressar o múltiplo P/E (*Price-to-Earnings*), crescimento de receita e Yield de Fluxo de Caixa Livre (*Free Cash Flow Yield*), reclassificando o risco do ativo de forma automática.
- **Termômetro de Risco de Bolha (CAPEX):** Painel dedicado para vigiar a correlação do CAPEX das Big Techs com a sustentabilidade do fluxo de caixa e demanda por GPUs.
- **Arquitetura Resiliente contra Cota / Quota Excedida:** Tratamento elegante contra instabilidades de rede e erros críticos `429 (Resource Exhausted/Quota Exceeded)`. Diante de tais adversidades, o sistema ativa imediatamente o banco de dados consolidado de referência para que a interface continue 100% interativa.
- **Gráficos Heurísticos Avançados:** Exibição intuitiva com Recharts comparando múltiplos de P/E vs. média de 5 anos e crescimento com momentum de geração de fluxo de caixa livre.
- **Console para Análise Inteligente Integrada:** Permite consultas qualitativas expressas para qualquer outro ativo de IA da S&P 500.

---

## 🛠️ Tecnosfera e Arquitetura

O app segue o ecossistema moderno Full-Stack (Express + React SPA com Vite):

- **Frontend:** React 19, TypeScript, Tailwind CSS, Lucide Icons, Recharts para visualizações, Plus Jakarta Sans & JetBrains Mono para tipografia técnica imersiva.
- **Backend (Express Server):** Implementa um proxy de backend resiliente sob portabilidade robusta para as chamadas da API Gemini SDK `@google/genai` (nunca expondo chaves secretas ao navegador no lado do cliente).
- **Estrutura dos Arquivos Criados/Modificados:**
  - `server.ts`: Ponto de entrada do backend. Proxy resiliente para chamadas e tratamento do motor de fallback de dados históricos de estresse.
  - `src/App.tsx`: Painel central imersivo contendo o console analítico, charts de distorção de preços e indicadores de pânico/ganância.
  - `src/types.ts`: Tipagem estrita de contratos de dados de mercado e grounding de pesquisa.
  - `src/components/MetricCard.tsx`: Cartão de perfilamento do ativo com simulador individual de valuation.
  - `src/components/KnowledgeBaseDoc.tsx`: Exposição legível da Base de Conhecimento e as estratégias estruturais de entrada e saída.

---

## 💎 Matriz de Decisão e Limites Quantitativos

A classificação atual dos ativos segue rigidamente as seguintes regras de conformidade:

- **Nvidia (NVDA):** Alerta crítico se ultrapassar 50x o P/E projetado (*Forward P/E*).
- **Palantir (PLTR):** Alerta de **Bolha Especulativa** se o P/E atual avançar além de 150x sem que o crescimento correspondente de receita acompanhe a aceleração de valor de mercado.
- **Magnificent 7 & Broadcom (AVGO):** Alinhados idealmente entre 25x e 35x. Acima de 40x exige sólida tese de geração de margens operacionais de IA.
- **Indicador de CAPEX:** A Nvidia e semicondutores correlacionados dependem diretamente da expansão conjunta do CAPEX de infraestrutura da MSFT, GOOGL, AMZN e META. Se houver retração, a Nvidia sofrerá colapso em cadeia.

Certifique-se de possuir o Node.js v18+.

npm install
